# Login Test

## Prerequisites
- User account with username "testuser" and password "password123" exists in the system

## Steps
1. Navigate to the login page at https://example.com/login
2. Enter "testuser" in the username field
3. Enter "password123" in the password field
4. Click the "Sign In" button
5. Verify that the user is redirected to the dashboard page
6. Verify that a welcome message with the username is displayed

## Expected Results
- User should be successfully logged in
- Dashboard page should show the user's name
- Session should be established correctly
